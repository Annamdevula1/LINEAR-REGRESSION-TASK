# LINEAR-REGRESSION-TASK
House Price Prediction using Linear Regression

Project Overview

This project predicts house prices using the Linear Regression algorithm in Machine Learning. The model is trained on a house price dataset and evaluated using RMSE (Root Mean Squared Error).

Tools Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

Dataset

House Price Dataset (data.csv)

Project Steps

1. Load the dataset.
2. Perform data preprocessing.
3. Select features and target variable.
4. Split data into training and testing sets.
5. Train the Linear Regression model.
6. Predict house prices.
7. Evaluate the model using RMSE.
8. Save predictions and trained model.

Results

- Successfully trained a Linear Regression model.
- Generated house price predictions.
- Calculated RMSE for model evaluation.
